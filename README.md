# CEP Primeiro

Site manifesto-interativo que defende que formulários brasileiros devem pedir o CEP antes de qualquer outro campo de endereço.

São 8 dígitos. 4 campos preenchidos automaticamente. Sem dropdown. Sem scroll.

## Demo

[felipebossolani.github.io/cep-primeiro](https://felipebossolani.github.io/cep-primeiro/)

## Stack

HTML + CSS + JS puros em um único `index.html`. Zero dependências externas.

## Créditos

- [AwesomeAPI](https://cep.awesomeapi.com.br) — API gratuita de consulta de CEP
